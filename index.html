User Interface Style Guide

User Interface Style Guide project aims to maintain a consistent front-end elements including buttons, forms and other elements using standard HTML/CSS markups in once place. You can make a number of changes to your existing style guide without uploading a new one. All UI elements would be updated once they are committed via GitHub.

Currently, this UI Style Guide project is under progress. Bookmark this page and come back later to see changes in place.

Fork this project on GitHub.
